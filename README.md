# Projeto - Introducao Aprendizado-de-Maquina
Projeto de senvolvido na disciplinaIntrodução Aprendizado de Máquina - 6º Semestre - ADS

**TEMA:** Análise e Previsão de Fatalidade em Acidentes de Trânsito

**INTEGRANTES:** Ester Cattaneo, Giovana Silveira Marques e Thais Yuri Hirata.

**INTRUÇÕES DE EXECUÇÃO:** Basta apenas adicionar a base de dados Denatran.csv na pasta .Content e executar os códigos.

**RESULTADOS PRINCIPAIS:** A Acurácia é irrelevante devido ao desbalanceamento de classes, o F1-Score da classe minoritária (fatalidades) é a métrica correta.
O modelo Random Forest Otimizado atingiu o melhor desempenho (F1-Score: 0.34), superando os demais modelos, seguido pela Regressão Logística que foi o segundo melhor modelo (F1-Score: 0.31).Apesar do melhor desempenho em relação aos demais modelos, ainda assim, a taxa de previsão está muito incerta e baixa.
